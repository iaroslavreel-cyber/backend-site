

Проект представляет собой учебный backend-сайт на Python + Flask.

Сайт имитирует небольшой образовательный сервис, на котором пользователь может посмотреть информацию о проекте, курсах, уроках и учебных материалах.


###  Главная страница

Система должна позволять пользователю открыть главную страницу сайта.

На главной странице должно отображаться:
- название сайта;
- краткое описание сайта;
- контактная информация;
- несколько отзывов учеников;
- ссылки на основные страницы сайта.

---

### . Страница "Курсы"

Система должна позволять пользователю открыть страницу со списком курсов.

На странице "Курсы" должно отображаться:
- название каждого курса;
- краткое описание каждого курса;
- ссылка для перехода на страницу конкретного курса.

Примеры курсов:
- математика;
- физика;
- информатика.

---

### FR-3. Страница отдельного курса

Система должна позволять пользователю открыть страницу отдельного курса.

На странице курса должно отображаться:
- название курса;
- краткое описание курса;
- список уроков;
- краткое описание каждого урока;
- ссылка на видеоурок;
- ссылка на файл с конспектом;
- ссылка на файл с домашним заданием.


---

###  Страница "О нас"

Система должна позволять пользователю открыть страницу "О нас".

На странице "О нас" должно отображаться:
- описание проекта;
- информация об образовательной деятельности;
- информация о сертификатах;
- информация о трудоустройстве или опыте преподавателей;
- дополнительная информация, повышающая доверие к сайту.

---

### Навигационное меню

Система должна отображать навигационное меню на страницах сайта.

Навигационное меню должно позволять пользователю перейти на основные страницы:
- главная страница;
- страница "Курсы";
- страница отдельного курса;
- страница "О нас".

Меню может располагаться сверху или сбоку страницы.

---




##  Минимальный ожидаемый результат

После реализации первой версии сайт должен содержать следующие страницы:

- `/` — главная страница;
- `/courses` — список курсов;
- `/courses/math` — страница курса по математике;
- `/about` — страница "О нас".

Все страницы должны открываться в браузере и быть доступны после деплоя.

На каждой странице должно быть меню для перехода между основными разделами сайта.
