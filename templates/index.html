Добро пожаловать

Это сайт онлайн-школы, созданной для учебного проекта. Пока здесь нет сложных возможностей, но постепенно сайт будет развиваться: появятся страницы курсов, уроки, материалы и полезная информация для учеников.

Для кого этот сайт?

Сайт предназначен для школьников, которые хотят повторить материал, разобраться в сложных темах и подготовиться к занятиям или экзаменам.

Какие курсы у нас есть?

  • Курс по математике
  • Курс по русскому языку
  • Курс по физике
  • Курс по информатике

Курс по математике

В этом курсе ученики смогут повторить основные темы школьной математики: уравнения, функции, графики, геометрию и задачи.

Курс по русскому языку

Курс поможет повторить правила орфографии, пунктуации, а также подготовиться к письменным заданиям.

Курс по физике

В курсе по физике будут разобраны основные темы: механика, электричество, оптика и простые задачи.

Курс по информатике

Этот курс будет посвящён основам программирования, алгоритмам и работе с данными.

Помощь
